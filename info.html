Dues X

Deus X is a custom, privately trained set of Neural Net weights. They were created by Albert Silver using the lc0 suite of tools. You can view more information at lczero.org.
The weights were trained exclusively using lc0 training code and run on the lc0 binary.

This Match

This match is a fun match to see if Deus X is capable of dethroning the might Stockfish.
It is played on the same hardware, same time control, and with same openings as a 40 game match with lc0 (11248 + 32072 playing with <= 16 pieces).
